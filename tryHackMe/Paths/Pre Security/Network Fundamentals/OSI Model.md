# OSI Model

![banner](imgs/OSI%20Model/roomBanner.png)

## Room objectives

- Open Systems Interconnection (OSI model).
  - Layer 7 : Application
  - Layer 6 : presentation
  - Layer 5 : session
  - Layer 4 : transport
  - Layer 3 : network
  - Layer 2 : Data link
  - Layer 1 : physical
- difference between TCP and UDP
